# Explainable-AI-Disease-Detection-through-Chest-X-Ray-using-Deep-Learning-and-XAI-visualization.
This project is an Explainable AI-based Disease Detection System using Chest X-ray images and Deep Learning. The system allows doctors to upload X-ray images, detect diseases such as pneumonia, view confidence percentages, and understand AI predictions through Explainable AI heatmaps and analysis.
